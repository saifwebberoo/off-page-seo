Boost your Off Page SEO with new WP plugin.

== Description ==

This plugin gives you lot of opportunities to explore possible backlinks, your competitors, it checks for your position in Google search results and much more.

== Installation ==

Installation is quite simple.

1) Download and install plugin.
2) Set up keywords in Settings.
3) Set up social metrics you want to follow.
4) Enjoy other tools!
